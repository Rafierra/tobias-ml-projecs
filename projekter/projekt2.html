Understanding Dynamic Mode Decomposition (DMD)

Introduction

Dynamic Mode Decomposition (DMD) is a powerful tool for analyzing complex, time-dependent systems from data. It separates the system into modes that evolve with fixed frequencies and growth/decay rates — helping us understand both spatial patterns and temporal dynamics.

How DMD Works

DMD assumes you have snapshots of a system over time: \( x_1, x_2, ..., x_m \), where each snapshot is a column vector.

We arrange the data into two matrices:

\[ X = [x_1, x_2, ..., x_{m-1}], \quad X' = [x_2, x_3, ..., x_m] \]

DMD finds a matrix \( A \) such that:

\[ X' \approx A X \]

We compute the best-fit linear operator and extract its eigenvalues and eigenvectors using SVD and low-rank projection.

Key Concepts

  • Modes: spatial patterns (eigenvectors of \( A \))
  • Eigenvalues: describe growth/decay and oscillation
  • Amplitudes: how much each mode contributes initially
  • Dynamics: time evolution of each mode
